# Image processing library written in cpp

just a basic implementation of some image processing utilites written in c++ for computer vision and AI/ML.

## features
- loading from csv files into matrix
- common matrix operations(add,sub,scaler multiplication, dot product, determinant, transpose)
- generating a .ppm file from matrix
- convolving simple filters(convolution is done without flipping the kernel)

## requirements
- .ppm file viewer to see the output image
- c++ compiler
## How to run
1. run `make` then execute main.exe
2. for windows simply run `build.bat`
